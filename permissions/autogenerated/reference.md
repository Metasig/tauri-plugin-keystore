## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-remove`
- `allow-contains-key`
- `allow-contains-unencrypted-key`
- `allow-retrieve`
- `allow-store`
- `allow-shared-secret`
- `allow-shared-secret-pub-key`
- `allow-store-unencrypted`
- `allow-retrieve-unencrypted`
- `allow-hmac-sha256`
